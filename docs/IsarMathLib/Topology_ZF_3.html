section ‚ÄπTopology 3‚Ä∫

theory Topology_ZF_3 imports Topology_ZF_2 FiniteSeq_ZF

begin

text‚Äπ‚ÄπTopology_ZF_1‚Ä∫ theory describes how we can define  a topology on a product
of two topological spaces. One way to generalize that is to construct topology for a cartesian
product of $n$ topological spaces. The cartesian product approach is somewhat inconvenient though.
Another way to approach product topology on $X^n$ is to model cartesian product as sets of 
sequences (of length $n$) of elements of $X$. This means that having a topology on $X$ we want
to define a toplogy on the space $n\rightarrow X$, where $n$ is a natural number (recall that 
$n = \{ 0,1,...,n-1\}$ in ZF). However, this in turn can be done more generally by defining a 
topology on any function space $I\rightarrow X$, where $I$ is any set of indices. This is what we 
do in this theory.‚Ä∫

subsection‚ÄπThe base of the product topology‚Ä∫

text‚ÄπIn this section we define the base of the product topology.‚Ä∫

text‚ÄπSuppose $\mathcal{X} = I\rightarrow \bigcup T$ is a space of functions from some index set $I$
to the carrier of a topology $T$. Then take a finite collection of open sets $W:N\rightarrow T$ 
indexed by $N\subseteq I$. We can define a subset of $\mathcal{X}$ that models the cartesian product of $W$.
‚Ä∫

definition
  "FinProd(ùí≥,W) ‚â° {x‚ààùí≥. ‚àÄ i‚ààdomain(W). x`(i) ‚àà W`(i)}"

text‚ÄπNow we define the base of the product topology as the collection of all finite products 
(in the sense defined above) of open sets. 
‚Ä∫

definition
  "ProductTopBase(I,T) ‚â°  ‚ãÉN‚ààFinPow(I).{FinProd(I‚Üí‚ãÉT,W). W‚ààN‚ÜíT}"

text‚ÄπFinally, we define the product topology on sequences. We use the ''Seq'' 
prefix although the definition is good for any index sets, not only natural numbers.‚Ä∫

definition
  "SeqProductTopology(I,T) ‚â° {‚ãÉB. B ‚àà Pow(ProductTopBase(I,T))}"

text‚ÄπProduct topology base is closed with respect to intersections.‚Ä∫

lemma prod_top_base_inter: 
  assumes A1: "T {is a topology}" and  
  A2: "U ‚àà ProductTopBase(I,T)"  "V ‚àà ProductTopBase(I,T)"
  shows "U‚à©V ‚àà ProductTopBase(I,T)"
proof -
  let ?ùí≥ = "I‚Üí‚ãÉT"  
  from A2 obtain N‚á©1  W‚á©1 N‚á©2 W‚á©2 where 
    I: "N‚á©1 ‚àà FinPow(I)"  "W‚á©1‚ààN‚á©1‚ÜíT"  "U = FinProd(?ùí≥,W‚á©1)" and
    II: "N‚á©2 ‚àà FinPow(I)"  "W‚á©2‚ààN‚á©2‚ÜíT"  "V = FinProd(?ùí≥,W‚á©2)"
    using ProductTopBase_def by auto
  let ?N‚á©3 = "N‚á©1 ‚à™ N‚á©2"
  let ?W‚á©3 = "{‚ü®i,if i ‚àà N‚á©1-N‚á©2 then W‚á©1`(i) 
        else if i ‚àà N‚á©2-N‚á©1 then  W‚á©2`(i) 
        else (W‚á©1`(i)) ‚à© (W‚á©2`(i))‚ü©. i ‚àà ?N‚á©3}"
  from A1 I II have "‚àÄi ‚àà N‚á©1 ‚à© N‚á©2.  (W‚á©1`(i) ‚à© W‚á©2`(i)) ‚àà T"
      using apply_funtype IsATopology_def by auto
  moreover from I II have "‚àÄi‚ààN‚á©1-N‚á©2. W‚á©1`(i) ‚àà T" and "‚àÄi‚ààN‚á©2-N‚á©1. W‚á©2`(i) ‚àà T" 
      using apply_funtype by auto
  ultimately have  "?W‚á©3:?N‚á©3‚ÜíT" by (rule fun_union_overlap)
  with I II have "FinProd(?ùí≥,?W‚á©3) ‚àà ProductTopBase(I,T)" using union_finpow ProductTopBase_def
    by auto
  moreover have "U‚à©V = FinProd(?ùí≥,?W‚á©3)"
  proof
    { fix x assume "x‚ààU" and "x‚ààV"
      with ‚ÄπU = FinProd(?ùí≥,W‚á©1)‚Ä∫  ‚ÄπW‚á©1‚ààN‚á©1‚ÜíT‚Ä∫ and  ‚ÄπV = FinProd(?ùí≥,W‚á©2)‚Ä∫  ‚ÄπW‚á©2‚ààN‚á©2‚ÜíT‚Ä∫
      have "x‚àà?ùí≥" and "‚àÄi‚ààN‚á©1. x`(i) ‚àà W‚á©1`(i)" and "‚àÄi‚ààN‚á©2. x`(i) ‚àà W‚á©2`(i)"
        using func1_1_L1 FinProd_def by auto
      with ‚Äπ?W‚á©3:?N‚á©3‚ÜíT‚Ä∫ ‚Äπx‚àà?ùí≥‚Ä∫ have "x ‚àà FinProd(?ùí≥,?W‚á©3)"  
        using ZF_fun_from_tot_val func1_1_L1 FinProd_def by auto
    } thus "U‚à©V ‚äÜ FinProd(?ùí≥,?W‚á©3)" by auto
    { fix x assume "x ‚àà FinProd(?ùí≥,?W‚á©3)"
      with ‚Äπ?W‚á©3:?N‚á©3‚ÜíT‚Ä∫ have "x:I‚Üí‚ãÉT" and III: "‚àÄi‚àà?N‚á©3. x`(i) ‚àà ?W‚á©3`(i)"
        using FinProd_def func1_1_L1 by auto
     { fix i assume "i‚ààN‚á©1" 
       with ‚Äπ?W‚á©3:?N‚á©3‚ÜíT‚Ä∫ have "?W‚á©3`(i) ‚äÜ W‚á©1`(i)" using ZF_fun_from_tot_val by auto
       with III ‚Äπi‚ààN‚á©1‚Ä∫ have "x`(i) ‚àà W‚á©1`(i)" by auto
     } with ‚ÄπW‚á©1‚ààN‚á©1‚ÜíT‚Ä∫ ‚Äπx:I‚Üí‚ãÉT‚Ä∫ ‚ÄπU = FinProd(?ùí≥,W‚á©1)‚Ä∫ 
      have "x ‚àà U" using func1_1_L1 FinProd_def by auto
      moreover
      { fix i assume "i‚ààN‚á©2" 
        with ‚Äπ?W‚á©3:?N‚á©3‚ÜíT‚Ä∫ have "?W‚á©3`(i) ‚äÜ W‚á©2`(i)" using ZF_fun_from_tot_val by auto
        with III ‚Äπi‚ààN‚á©2‚Ä∫ have "x`(i) ‚àà W‚á©2`(i)" by auto
      } with ‚ÄπW‚á©2‚ààN‚á©2‚ÜíT‚Ä∫ ‚Äπx:I‚Üí‚ãÉT‚Ä∫ ‚ÄπV = FinProd(?ùí≥,W‚á©2)‚Ä∫ have "x‚ààV" 
          using func1_1_L1 FinProd_def by auto 
      ultimately have "x ‚àà U‚à©V" by simp
    } thus "FinProd(?ùí≥,?W‚á©3) ‚äÜ U‚à©V" by auto
  qed
    ultimately show ?thesis by simp
qed 

text‚ÄπIn the next theorem we show the collection of sets defined above as 
‚ÄπProductTopBase(ùí≥,T)‚Ä∫ satisfies the base condition. This is a condition, defined in 
‚ÄπTopology_ZF_1‚Ä∫ that allows to claim that this collection is a base for some topology.‚Ä∫

theorem prod_top_base_is_base: assumes "T {is a topology}" 
  shows "ProductTopBase(I,T) {satisfies the base condition}"
  using assms prod_top_base_inter inter_closed_base by simp

text‚ÄπThe (sequence) product topology is indeed a topology on the space of sequences.
In the proof we are using the fact that $(\emptyset\rightarrow X) = \{\emptyset\}$.‚Ä∫

theorem seq_prod_top_is_top:  assumes "T {is a topology}"
  shows 
  "SeqProductTopology(I,T) {is a topology}" and 
  "ProductTopBase(I,T) {is a base for} SeqProductTopology(I,T)" and
  "‚ãÉSeqProductTopology(I,T) = (I‚Üí‚ãÉT)"
proof -
  from assms show "SeqProductTopology(I,T) {is a topology}" and 
    I: "ProductTopBase(I,T) {is a base for} SeqProductTopology(I,T)"
      using prod_top_base_is_base SeqProductTopology_def Top_1_2_T1
        by auto
  from I have "‚ãÉSeqProductTopology(I,T) = ‚ãÉProductTopBase(I,T)"
    using Top_1_2_L5 by simp
  also have "‚ãÉProductTopBase(I,T) = (I‚Üí‚ãÉT)"
  proof
    show "‚ãÉProductTopBase(I,T) ‚äÜ (I‚Üí‚ãÉT)" using ProductTopBase_def FinProd_def
      by auto
    have "0 ‚àà FinPow(I)" using empty_in_finpow by simp
    hence "{FinProd(I‚Üí‚ãÉT,W). W‚àà0‚ÜíT} ‚äÜ (‚ãÉN‚ààFinPow(I).{FinProd(I‚Üí‚ãÉT,W). W‚ààN‚ÜíT})"
      by blast
    then show "(I‚Üí‚ãÉT) ‚äÜ ‚ãÉProductTopBase(I,T)" using ProductTopBase_def FinProd_def
      by auto
  qed     
  finally show "‚ãÉSeqProductTopology(I,T) = (I‚Üí‚ãÉT)" by simp
qed

subsection‚ÄπFinite product of topologies‚Ä∫

text‚ÄπAs a special case of the space of functions $I\rightarrow X$ we can consider space of lists
of elements of $X$, i.e. space $n\rightarrow X$, where $n$ is a natural number (recall that in ZF 
set theory $n=\{0,1,...,n-1\}$). Such spaces model finite cartesian products $X^n$ 
but are easier to deal with in formalized way (than the said products). 
This section discusses natural topology defined on $n\rightarrow X$ where $X$ is a topological space.
‚Ä∫

text‚ÄπWhen the index set is finite, the definition of ‚ÄπProductTopBase(I,T)‚Ä∫ 
can be simplifed.‚Ä∫

lemma fin_prod_def_nat: assumes A1: "n‚àànat" and A2: "T {is a topology}"  
  shows "ProductTopBase(n,T) = {FinProd(n‚Üí‚ãÉT,W). W‚ààn‚ÜíT}"
proof
  from A1 have "n ‚àà FinPow(n)" using nat_finpow_nat fin_finpow_self by auto
  then show "{FinProd(n‚Üí‚ãÉT,W). W‚ààn‚ÜíT} ‚äÜ ProductTopBase(n,T)" using ProductTopBase_def
    by auto
  { fix B assume "B ‚àà ProductTopBase(n,T)"
    then obtain N W where  "N ‚àà FinPow(n)" and "W ‚àà N‚ÜíT" and "B = FinProd(n‚Üí‚ãÉT,W)"
      using ProductTopBase_def by auto
    let ?W‚á©n = "{‚ü®i,if i‚ààN then W`(i) else ‚ãÉT‚ü©. i‚ààn}"
    from A2  ‚ÄπN ‚àà FinPow(n)‚Ä∫  ‚ÄπW‚ààN‚ÜíT‚Ä∫ have "‚àÄi‚ààn. (if i‚ààN then W`(i) else ‚ãÉT) ‚àà T"
      using apply_funtype FinPow_def IsATopology_def by auto
    then have "?W‚á©n:n‚ÜíT" by (rule ZF_fun_from_total)
    moreover have "B = FinProd(n‚Üí‚ãÉT,?W‚á©n)"
    proof
      { fix x assume "x‚ààB"
        with ‚ÄπB = FinProd(n‚Üí‚ãÉT,W)‚Ä∫ have "x ‚àà n‚Üí‚ãÉT" using FinProd_def by simp
        moreover have "‚àÄi‚ààdomain(?W‚á©n). x`(i) ‚àà ?W‚á©n`(i)"
        proof
          fix i assume "i ‚àà domain(?W‚á©n)"
          with ‚Äπ?W‚á©n:n‚ÜíT‚Ä∫ have "i‚ààn" using func1_1_L1 by simp 
          with ‚Äπx:n‚Üí‚ãÉT‚Ä∫ have "x`(i) ‚àà ‚ãÉT" using apply_funtype by blast
          with ‚Äπx‚ààB‚Ä∫ ‚ÄπB = FinProd(n‚Üí‚ãÉT,W)‚Ä∫ ‚ÄπW ‚àà N‚ÜíT‚Ä∫ ‚Äπ?W‚á©n:n‚ÜíT‚Ä∫ ‚Äπi‚ààn‚Ä∫
          show "x`(i) ‚àà ?W‚á©n`(i)" using func1_1_L1 FinProd_def ZF_fun_from_tot_val 
            by simp
        qed
        ultimately have "x ‚àà FinProd(n‚Üí‚ãÉT,?W‚á©n)" using FinProd_def by simp
      } thus "B ‚äÜ FinProd(n‚Üí‚ãÉT,?W‚á©n)" by auto
      next
      { fix x assume "x ‚àà FinProd(n‚Üí‚ãÉT,?W‚á©n)"    
        then have "x ‚àà n‚Üí‚ãÉT" and "‚àÄi‚ààdomain(?W‚á©n). x`(i) ‚àà ?W‚á©n`(i)" 
          using  FinProd_def by auto
        with ‚Äπ?W‚á©n:n‚ÜíT‚Ä∫ and ‚ÄπN ‚àà FinPow(n)‚Ä∫ have "‚àÄi‚ààN. x`(i) ‚àà ?W‚á©n`(i)"
          using func1_1_L1 FinPow_def by auto
        moreover from ‚Äπ?W‚á©n:n‚ÜíT‚Ä∫ and ‚ÄπN ‚àà FinPow(n)‚Ä∫ 
        have "‚àÄi‚ààN. ?W‚á©n`(i) = W`(i)"
          using ZF_fun_from_tot_val FinPow_def by auto
        ultimately have "‚àÄi‚ààN. x`(i) ‚àà W`(i)" by simp
        with ‚ÄπW ‚àà N‚ÜíT‚Ä∫ ‚Äπx ‚àà n‚Üí‚ãÉT‚Ä∫ ‚ÄπB = FinProd(n‚Üí‚ãÉT,W)‚Ä∫ have "x‚ààB"
          using func1_1_L1 FinProd_def by simp
     } thus "FinProd(n‚Üí‚ãÉT,?W‚á©n) ‚äÜ B" by auto
  qed 
    ultimately have "B ‚àà {FinProd(n‚Üí‚ãÉT,W). W‚ààn‚ÜíT}" by auto
  } thus "ProductTopBase(n,T) ‚äÜ {FinProd(n‚Üí‚ãÉT,W). W‚ààn‚ÜíT}" by auto
qed

text‚ÄπA technical lemma providing a formula for finite product on one topological space.‚Ä∫

lemma single_top_prod: assumes A1: "W:1‚ÜíœÑ" 
  shows "FinProd(1‚Üí‚ãÉœÑ,W) = { {‚ü®0,y‚ü©}. y ‚àà W`(0)}"
proof -
  have "1 = {0}" by auto
  from A1 have "domain(W) = {0}" using func1_1_L1 by auto
  then have "FinProd(1‚Üí‚ãÉœÑ,W) = {x ‚àà 1‚Üí‚ãÉœÑ. x`(0) ‚àà W`(0)}"
    using FinProd_def by simp
  also have "{x ‚àà 1‚Üí‚ãÉœÑ. x`(0) ‚àà W`(0)} = { {‚ü®0,y‚ü©}. y ‚àà W`(0)}"
  proof
    from ‚Äπ1 = {0}‚Ä∫ show "{x ‚àà 1‚Üí‚ãÉœÑ. x`(0) ‚àà W`(0)} ‚äÜ { {‚ü®0,y‚ü©}. y ‚àà W`(0)}"
      using func_singleton_pair by auto 
    { fix x assume "x ‚àà { {‚ü®0,y‚ü©}. y ‚àà W`(0)}"
      then obtain y where "x = {‚ü®0,y‚ü©}" and II: "y ‚àà W`(0)" by auto
      with A1 have "y ‚àà ‚ãÉœÑ" using apply_funtype by auto
      with ‚Äπx = {‚ü®0,y‚ü©}‚Ä∫  ‚Äπ1 = {0}‚Ä∫ have "x:1‚Üí‚ãÉœÑ" using pair_func_singleton
        by auto
      with ‚Äπx = {‚ü®0,y‚ü©}‚Ä∫ II have "x ‚àà {x ‚àà 1‚Üí‚ãÉœÑ. x`(0) ‚àà W`(0)}"
        using pair_val by simp
    } thus "{ {‚ü®0,y‚ü©}. y ‚àà W`(0)} ‚äÜ {x ‚àà 1‚Üí‚ãÉœÑ. x`(0) ‚àà W`(0)}" by auto
  qed
  finally show ?thesis by simp 
qed

text‚ÄπIntuitively, the topological space of singleton lists valued in $X$ 
  is the same as $X$. However, each element of this space is a list of length one,
  i.e a set consisting of a pair $\langle 0, x\rangle$ where $x$ is an element of $X$.
  The next lemma provides a formula for the product topology in the corner case when we have 
  only one factor and shows that the product topology of one space is essentially the same as 
  the space.‚Ä∫

lemma singleton_prod_top: assumes A1: "œÑ {is a topology}" 
  shows 
    "SeqProductTopology(1,œÑ) = { { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ}" and
    "IsAhomeomorphism(œÑ,SeqProductTopology(1,œÑ),{‚ü®y,{‚ü®0,y‚ü©}‚ü©.y ‚àà ‚ãÉœÑ})"
proof -
  have "{0} = 1" by auto
  let ?b = "{‚ü®y,{‚ü®0,y‚ü©}‚ü©.y ‚àà ‚ãÉœÑ}"
  have "?b ‚àà bij(‚ãÉœÑ,1‚Üí‚ãÉœÑ)" using list_singleton_bij by blast
  with A1 have "{?b``(U). U‚ààœÑ} {is a topology}" and "IsAhomeomorphism(œÑ, {?b``(U). U‚ààœÑ},?b)"
    using bij_induced_top by auto
  moreover have "‚àÄU‚ààœÑ. ?b``(U) = { {‚ü®0,y‚ü©}. y‚ààU }"
  proof
    fix U assume "U‚ààœÑ"
    from ‚Äπ?b ‚àà bij(‚ãÉœÑ,1‚Üí‚ãÉœÑ)‚Ä∫ have "?b:‚ãÉœÑ‚Üí(1‚Üí‚ãÉœÑ)" using bij_def inj_def
      by simp
    { fix y assume "y ‚àà ‚ãÉœÑ"
      with ‚Äπ?b:‚ãÉœÑ‚Üí(1‚Üí‚ãÉœÑ)‚Ä∫ have "?b`(y) = {‚ü®0,y‚ü©}" using ZF_fun_from_tot_val
        by simp
    } hence "‚àÄy ‚àà ‚ãÉœÑ. ?b`(y) = {‚ü®0,y‚ü©}" by auto
    with  ‚ÄπU‚ààœÑ‚Ä∫ ‚Äπ?b:‚ãÉœÑ‚Üí(1‚Üí‚ãÉœÑ)‚Ä∫ show " ?b``(U) = { {‚ü®0,y‚ü©}. y‚ààU }"
      using func_imagedef by auto 
  qed
  moreover have "ProductTopBase(1,œÑ) = { { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ}"
  proof
    { fix V assume "V ‚àà ProductTopBase(1,œÑ)"
      with A1 obtain W where "W:1‚ÜíœÑ" and "V = FinProd(1‚Üí‚ãÉœÑ,W)"
        using fin_prod_def_nat by auto
      then have "V ‚àà { { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ}" using apply_funtype single_top_prod
        by auto 
    } thus "ProductTopBase(1,œÑ) ‚äÜ { { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ}" by auto
  { fix V assume "V ‚àà { { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ}"
    then obtain U where "U‚ààœÑ" and "V = { {‚ü®0,y‚ü©}. y‚ààU }" by auto 
    let ?W = "{‚ü®0,U‚ü©}"
    from ‚ÄπU‚ààœÑ‚Ä∫ have "?W:{0}‚ÜíœÑ" using pair_func_singleton by simp   
    with ‚Äπ{0} = 1‚Ä∫ have "?W:1‚ÜíœÑ" and "?W`(0) = U" using pair_val by auto
    with ‚ÄπV = { {‚ü®0,y‚ü©}. y‚ààU }‚Ä∫ have "V = FinProd(1‚Üí‚ãÉœÑ,?W)"
      using single_top_prod by simp 
    with A1 ‚Äπ?W:1‚ÜíœÑ‚Ä∫ have "V ‚àà ProductTopBase(1,œÑ)" using fin_prod_def_nat
      by auto
   } thus "{ { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ} ‚äÜ ProductTopBase(1,œÑ)" by auto 
  qed
  ultimately have I: "ProductTopBase(1,œÑ) {is a topology}" and 
    II: "IsAhomeomorphism(œÑ, ProductTopBase(1,œÑ),?b)" by auto
  from A1 have "ProductTopBase(1,œÑ) {is a base for} SeqProductTopology(1,œÑ)" 
    using seq_prod_top_is_top by simp
  with I have "ProductTopBase(1,œÑ) = SeqProductTopology(1,œÑ)" by (rule base_topology)
  with ‚ÄπProductTopBase(1,œÑ) = { { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ}‚Ä∫ II show
    "SeqProductTopology(1,œÑ) = { { {‚ü®0,y‚ü©}. y‚ààU }. U‚ààœÑ}" and
    "IsAhomeomorphism(œÑ,SeqProductTopology(1,œÑ),{‚ü®y,{‚ü®0,y‚ü©}‚ü©.y ‚àà ‚ãÉœÑ})" by auto
qed

text‚ÄπA special corner case of ‚Äπfinite_top_prod_homeo‚Ä∫: a space $X$ 
  is homeomorphic to the space of one element lists of $X$.‚Ä∫

theorem singleton_prod_top1: assumes A1: "œÑ {is a topology}"
