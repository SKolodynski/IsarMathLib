section ‚Äπ Real valued metric spaces ‚Ä∫

theory MetricSpace_ZF_1 imports Real_ZF_2
begin

text‚ÄπThe development of metric spaces in IsarMathLib is different from the usual treatment
  of the subject because the notion of a metric (or a pseudometric) is defined in the 
  ‚ÄπMetricSpace_ZF‚Ä∫ theory a more generally as a function valued in an ordered loop. 
  This theory file brings the subject closer to the standard way by specializing that general
  definition to the usual special case where the value of the metric are nonnegative real numbers. ‚Ä∫

subsection‚ÄπReal valued metric scapes: context and notation‚Ä∫

text‚Äπ The ‚Äπreals‚Ä∫ context (locale) defined in the ‚ÄπReal_ZF_2‚Ä∫ theory fixes a model of reals 
  (i.e. a complete ordered field) and defines notation for things like zero, one, the set of 
  positive numbers, absolute value etc. For metric spaces we reuse the notation defined there.‚Ä∫

text‚ÄπThe ‚Äπrpmetric_space‚Ä∫ locale extends the ‚Äπreals‚Ä∫ locale, adding the carrier $X$ 
  of the metric space and the metric $\mathcal{d}$ to the context, together with the assumption
  that $\mathcal{d}:X\times X \rightarrow \mathbb{R}^+$ is a pseudo metric.
  We choose to denote the disk in $X$ with center $c$ and radius $r$ as ‚Äπball(c,r)‚Ä∫ 
  As in the ‚Äπpmetric_space‚Ä∫ locale we define the $\tau$ to be the metric topology, i.e.
  the topology induced by the (real valued) pseudometric $\mathcal{d}$.
  An alternative would be to define the ‚Äπrpmetric_space‚Ä∫ as an extension of the ‚Äπrpmetric_space‚Ä∫
  context, but that is in turn an extension of the ‚Äπloop1‚Ä∫ locale that defines notation
  for left and right division which which do not want in the context of real numbers. ‚Ä∫

locale rpmetric_space = reals +
  fixes X and ùñΩ
  assumes pmetricAssum: "IsApseudoMetric(ùñΩ,X,‚Ñù,Add,ROrd)"
  fixes ball
  defines ball_def [simp]: "ball(c,r) ‚â° Disk(X,ùñΩ,ROrd,c,r)"
  fixes pmettop ("œÑ") 
  defines pmettop [simp]: "œÑ ‚â° MetricTopology(X,‚Ñù,Add,ROrd,ùñΩ)"
  fixes interior ("int")
  defines interior_def [simp]: "int(D) ‚â° Interior(D,œÑ)"
  fixes cl
  defines cl_def [simp]: "cl(D) ‚â° Closure(D,œÑ)"

text‚ÄπThe propositions proven in the ‚Äπpmetric_space‚Ä∫ context defined in ‚ÄπMetric_Space_ZF‚Ä∫ theory 
  are valid in the ‚Äπrpmetric_space‚Ä∫ context. ‚Ä∫

lemma (in rpmetric_space) pmetric_space_rpmetric_space_valid: 
  shows "pmetric_space(‚Ñù,Add,ROrd,ùñΩ,X)"
  unfolding pmetric_space_def pmetric_space_axioms_def loop1_def
  using pmetricAssum reals_loop by simp

text‚ÄπThe context ‚Äπrpmetric_space‚Ä∫ is a special case of context ‚Äπpmetric_space‚Ä∫ 
  where the fixed objects in ‚Äπpmetric_space‚Ä∫ map to (in the order defined in ‚Äπpmetric_space‚Ä∫) 
  the set of real numbers, real addition, the order relation on reals, 
  the strict order relation on reals, the set of non-negative reals and 
  the set of positive reals. The metrics $d$ maps to the real metrics 
  ‚ÄπùñΩ‚Ä∫, the carrier of the metric space $X$ is still $X$, and the ‚Äπdisk‚Ä∫s from ‚Äπpmetric_space‚Ä∫
  are now called ‚Äπball‚Ä∫s in ‚Äπrpmetric_space‚Ä∫. The notation for right and left division from 
  ‚Äπrpmetric_space‚Ä∫ is not used in ‚Äπpmetric_space‚Ä∫. ‚Ä∫

  "‚Ñù" Add ROrd "ùü¨" realadd lesseq sless nonnegative positiveset
  "Œªx y. LeftDiv(‚Ñù,Add)`‚ü®x,y‚ü©"

text‚ÄπThe ‚Äπrmetric_space‚Ä∫ locale (context) specializes the the ‚Äπrpmetric_space‚Ä∫ context
  by adding the assumption of identity of indiscernibles. ‚Ä∫

text‚ÄπThe propositions proven in the ‚Äπmetric_space‚Ä∫ context defined in ‚ÄπMetric_Space_ZF‚Ä∫ theory 
  are valid in the ‚Äπrmetric_space‚Ä∫ context. ‚Ä∫

  shows "metric_space(‚Ñù,Add,ROrd,ùñΩ,X)"

  using pmetric_space_rpmetric_space_valid ident_indisc
  by simp

text‚ÄπThe ‚Äπrmetric_space‚Ä∫ context is a special case of the ‚Äπmetric_space‚Ä∫ context,
  with fixed objects mapping the same as in the mapping between ‚Äπrpmetric_space‚Ä∫
  and ‚Äπpmetric_space‚Ä∫ above. ‚Ä∫

subsection‚ÄπReal valued metric spaces are Hausdorff as topological spaces‚Ä∫

text‚ÄπThe usual (real-valued) metric spaces are a special case of ordered loop valued
  metric spaces defined in the ‚ÄπMetricSpace_ZF‚Ä∫ theory, hence they are $T_2$ 
  as topological spaces. Below we repeat the major theorems of ‚ÄπMetricSpace_ZF‚Ä∫ theory
  specialized the standard setting of real valued metrics. ‚Ä∫

text‚ÄπSince in the ‚Äπrpmetric_space‚Ä∫ context $\mathfrak{d}$ is a pseudometrics
  the (real valued) metric topology indeed a topology. ‚Ä∫

  shows  "œÑ {is a topology}"
  using rord_down_directs pmetric_is_top by simp

text‚ÄπThe collection of open disks (caled ‚Äπball‚Ä∫s in the ‚Äπrpmetric_space‚Ä∫ context
  is a base for the (real valued) metric topology.‚Ä∫

theorem (in rpmetric_space) rdisks_are_base:
  shows "(‚ãÉc‚ààX. {ball(c,R). R‚àà‚Ñù‚á©+}) {is a base for} œÑ"
  using rord_down_directs disks_are_base by simp

text‚Äπ$X$ is the carrier of the (real valued) metric topology.‚Ä∫

  using rord_down_directs metric_top_carrier by simp

subsection‚ÄπReal valued (pseudo)metric spaces as uniform spaces‚Ä∫

text‚ÄπThe ordered loop valued pseudometric spaces are uniform spaces. In this 
  section we specialize major propositions from that context to the real valued pseudometric. ‚Ä∫

text‚ÄπIn the ‚ÄπMetricSpace_ZF‚Ä∫ theory we define a property ‚ÄπIsHalfable‚Ä∫ of an ordered loop
  that states that for every positive element $b_1$ of the loop there is another (positive) 
  one $b_2$ such that $b_2+b_2 \leq b_1$. This property is needed for the ordered loop valued 
  pseudometric space to be a uniform space. In the next lemma we show that real numbers satisfy 
  this property. ‚Ä∫

lemma (in reals) pos_reals_halfable: shows "IsHalfable(‚Ñù,Add,ROrd)"
proof -
  { fix x assume "x‚àà‚Ñù‚á©+"
    let ?y = "(ùüÆ¬Ø)‚ãÖx"
    from ‚Äπx‚àà‚Ñù‚á©+‚Ä∫ have "x‚àà‚Ñù" and "?y‚àà‚Ñù‚á©+"
      using element_pos pos_mul_closed ord_ring_less_members one_half_pos(2)
      by simp_all
    from ‚Äπx‚àà‚Ñù‚Ä∫ have "(ùüÆ¬Ø\<ra>ùüÆ¬Ø)‚ãÖx = x" using half_half_one(2) Ring_ZF_1_L3(6) by simp
    with ‚Äπx‚àà‚Ñù‚Ä∫ have "?y\<ra>?y \<lsq> x"
      using ord_ring_less_members ring_oper_distr(2) one_half_pos(2) ring_ord_refl
      by auto
    with ‚Äπ?y‚àà‚Ñù‚á©+‚Ä∫ have "‚àÉy‚àà‚Ñù‚á©+. y \<ra> y \<lsq> x" by auto
  } then show ?thesis unfolding IsHalfable_def by simp
qed

text‚ÄπIn the ‚Äπrpmetric_space‚Ä∫ we will write ‚ÄπUniformGauge(X,‚Ñù,Add,ROrd,ùñΩ)‚Ä∫ i.e.
  $\{\mathcal{d}^{-1}([0,b]: b \in \mathbb{R}_+ \}$ as $\mathfrak{U}$. ‚Ä∫

abbreviation (in rpmetric_space) rgauge ("ùîò") where "ùîò ‚â° UniformGauge(X,‚Ñù,Add,ROrd,ùñΩ)"

text‚Äπ$\mathfrak{U}$ is a fundamental system of entourages, hence its supersets 
  form a uniformity on $X$ and hence those supersets define a topology on $X$.
  This is a special case of the theorem ‚Äπmetric_gauge_base‚Ä∫ from the ‚ÄπMetric_Space_ZF‚Ä∫ 
  theory but instead an ordered loop we have real numbers, so all the premises are automatically
  satisfied, except for the one of $X$ being nonempty.‚Ä∫

theorem (in rpmetric_space) metric_gauge_base: assumes "X‚â†‚àÖ"
  shows
    "ùîò {is a uniform base on} X"
    "Supersets(X√óX,ùîò) {is a uniformity on} X"
    "UniformTopology(Supersets(X√óX,ùîò),X) {is a topology}"
    "‚ãÉUniformTopology(Supersets(X√óX,ùîò),X) = X"
  using assms rord_down_directs pos_non_empty pos_reals_halfable metric_gauge_base
  by simp_all

text‚ÄπThe topology generated by open disks is the same as the one 
  coming from the the unifomity consisting of supersets of sets in $\mathfrak{U}$. ‚Ä∫

theorem (in rpmetric_space) rmetric_top_is_uniform_top: assumes "X‚â†‚àÖ"
  shows "œÑ = UniformTopology(Supersets(X√óX,ùîò),X)"
  using assms rord_down_directs pos_non_empty pos_reals_halfable metric_top_is_uniform_top
  by simp
