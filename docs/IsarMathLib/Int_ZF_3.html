text‚ÄπWe know  from ‚ÄπGroup_ZF_3.thy‚Ä∫ that finite range functions are almost homomorphisms. 
  Besides reminding that fact for slopes the next lemma shows 
  that finite range functions do not belong to ‚ÄπùíÆ‚á©+‚Ä∫. 
  This is important, because the projection
  of the set of finite range functions defines zero in the real number construction in ‚ÄπReal_ZF_x.thy‚Ä∫ 
  series, while the projection of ‚ÄπùíÆ‚á©+‚Ä∫ becomes the set of (strictly) positive reals. 
  We don't want zero to be positive, do we? The next lemma is a part of Lemma 5 in the Arthan's paper 
  \cite{Arthan2004}.‚Ä∫

lemma (in int1) Int_ZF_2_3_L1B: 
  assumes A1: "f ‚àà FinRangeFunctions(‚Ñ§,‚Ñ§)"
  shows "f‚ààùíÆ"   "f ‚àâ ùíÆ‚á©+"

  from A1 show "f‚ààùíÆ" using Int_ZF_2_1_L1 group1.Group_ZF_3_3_L1
    by auto
  have "‚Ñ§‚á©+ ‚äÜ ‚Ñ§" using PositiveSet_def by auto
  with A1 have "f``(‚Ñ§‚á©+) ‚àà Fin(‚Ñ§)"
    using Finite1_L21 by simp
  then have "f``(‚Ñ§‚á©+) ‚à© ‚Ñ§‚á©+ ‚àà Fin(‚Ñ§)"
    using Fin_subset_lemma by blast
  thus "f ‚àâ ùíÆ‚á©+" by auto
qed
