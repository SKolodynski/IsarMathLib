subsection‚ÄπDefinition and basic properties‚Ä∫

text‚ÄπIn this section we define what is a field and list the basic properties
  of fields.‚Ä∫

text‚ÄπField is a notrivial commutative ring such that all 
  non-zero elements have an inverse. We define the notion of being a field as
  a statement about three sets. The first set, denoted ‚ÄπK‚Ä∫ is the 
  carrier of the field. The second set, denoted ‚ÄπA‚Ä∫ represents the 
  additive operation on ‚ÄπK‚Ä∫ (recall that in ZF set theory functions 
  are sets). The third set ‚ÄπM‚Ä∫ represents the multiplicative operation 
  on ‚ÄπK‚Ä∫.‚Ä∫

definition
  "IsAfield(K,A,M) ‚â° 
  (IsAring(K,A,M) ‚àß (M {is commutative on} K) ‚àß
  TheNeutralElement(K,A) ‚â† TheNeutralElement(K,M) ‚àß 
  (‚àÄa‚ààK. a‚â†TheNeutralElement(K,A)‚ü∂
  (‚àÉb‚ààK. M`‚ü®a,b‚ü© = TheNeutralElement(K,M))))"

text‚ÄπThe ‚Äπfield0‚Ä∫ context extends the ‚Äπring0‚Ä∫
  context adding field-related assumptions and notation related to the 
  multiplicative inverse.‚Ä∫

  assumes mult_commute: "M {is commutative on} K"
  
  assumes not_triv: "ùü¨ ‚â† ùü≠"

  assumes inv_exists: "‚àÄa‚ààK. a‚â†ùü¨ ‚ü∂ (‚àÉb‚ààK. a‚ãÖb = ùü≠)"

  fixes non_zero ("K‚á©0")
  defines non_zero_def[simp]: "K‚á©0 ‚â° K-{ùü¨}"

  fixes inv ("_¬Ø " [96] 97)
  defines inv_def[simp]: "a¬Ø ‚â° GroupInv(K‚á©0,restrict(M,K‚á©0√óK‚á©0))`(a)"

subsection‚Äπ1/0=0‚Ä∫

text‚ÄπIn ZF if $f: X\rightarrow Y$ and $x\notin X$ we have $f(x)=\emptyset$.
  Since $\emptyset$ (the empty set) in ZF is the same as zero of natural numbers we
  can claim that $1/0=0$ in certain sense. In this section we prove a theorem that
  makes makes it explicit.‚Ä∫

text‚ÄπThe next locale extends the ‚Äπfield0‚Ä∫ locale to introduce notation
  for division operation.‚Ä∫

locale fieldd = field0 +
  fixes division
  defines division_def[simp]: "division ‚â° {‚ü®p,fst(p)‚ãÖsnd(p)¬Ø‚ü©. p‚ààK√óK‚á©0}"

  fixes fdiv (infixl "\<fd>" 95)
  defines fdiv_def[simp]: "x\<fd>y ‚â° division`‚ü®x,y‚ü©"


text‚ÄπDivision is a function on $K\times K_0$ with values in $K$.‚Ä∫

lemma (in fieldd) div_fun: shows "division: K√óK‚á©0 ‚Üí K"
proof -
  have "‚àÄp ‚àà K√óK‚á©0. fst(p)‚ãÖsnd(p)¬Ø ‚àà K"
  proof
    fix p assume "p ‚àà K√óK‚á©0"
    hence "fst(p) ‚àà K" and "snd(p) ‚àà K‚á©0" by auto
    then show  "fst(p)‚ãÖsnd(p)¬Ø ‚àà K" using Ring_ZF_1_L4 Field_ZF_1_L5 by auto
  qed
  then have "{‚ü®p,fst(p)‚ãÖsnd(p)¬Ø‚ü©. p‚ààK√óK‚á©0}: K√óK‚á©0 ‚Üí K"
    by (rule ZF_fun_from_total)
  thus ?thesis by simp
qed

  Isabelle's ‚Äπfunc.thy‚Ä∫.‚Ä∫

theorem (in fieldd) one_over_zero: shows "ùü≠\<fd>ùü¨ = 0"
proof-
  have "domain(division) = K√óK‚á©0" using div_fun func1_1_L1
    by simp
  hence "‚ü®ùü≠,ùü¨‚ü© ‚àâ domain(division)" by auto
  then show ?thesis using apply_0 by simp
qed
