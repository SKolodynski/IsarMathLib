subsection‚ÄπDefinition and basic properties‚Ä∫

text‚ÄπHere we define ordered fields and proove their basic properties.‚Ä∫

text‚ÄπOrdered field is a notrivial ordered ring such that all 
  non-zero elements have an inverse. We define the notion of being a ordered 
  field as
  a statement about four sets. The first set, denoted ‚ÄπK‚Ä∫ is the 
  carrier of the field. The second set, denoted ‚ÄπA‚Ä∫ represents the 
  additive operation on ‚ÄπK‚Ä∫ (recall that in ZF set theory functions 
  are sets). The third set ‚ÄπM‚Ä∫ represents the multiplicative operation 
  on ‚ÄπK‚Ä∫. The fourth set ‚Äπr‚Ä∫ is the order 
  relation on ‚ÄπK‚Ä∫.‚Ä∫

definition
  "IsAnOrdField(K,A,M,r) ‚â° (IsAnOrdRing(K,A,M,r) ‚àß
  (M {is commutative on} K) ‚àß
  TheNeutralElement(K,A) ‚â† TheNeutralElement(K,M) ‚àß
  (‚àÄa‚ààK. a‚â†TheNeutralElement(K,A)‚ü∂
  (‚àÉb‚ààK. M`‚ü®a,b‚ü© = TheNeutralElement(K,M))))"

text‚ÄπThe next context (locale) defines notation used for ordered fields. 
  We do that by extending the notation defined in the 
  ‚Äπring1‚Ä∫ context that is used for ordered rings and 
  adding some assumptions to make sure we are 
  talking about ordered fields in this context. 
  We should rename the carrier from $R$ used in the ‚Äπring1‚Ä∫ 
  context to $K$, more appriopriate for fields. Theoretically the Isar locale
  facility supports such renaming, but we experienced diffculties using
  some lemmas from ‚Äπring1‚Ä∫ locale after renaming. 
‚Ä∫

locale field1 = ring1 +

  assumes mult_commute: "M {is commutative on} R"
  
  assumes not_triv: "ùü¨ ‚â† ùü≠"

  assumes inv_exists: "‚àÄa‚ààR. a‚â†ùü¨ ‚ü∂ (‚àÉb‚ààR. a‚ãÖb = ùü≠)"

  fixes non_zero ("R‚á©0")
  defines non_zero_def[simp]: "R‚á©0 ‚â° R-{ùü¨}"

  fixes inv ("_¬Ø " [96] 97)
  defines inv_def[simp]: "a¬Ø ‚â° GroupInv(R‚á©0,restrict(M,R‚á©0√óR‚á©0))`(a)"

subsection‚ÄπDefinition of real numbers‚Ä∫

text‚ÄπThe only purpose of this section is to define what does it mean
  to be a model of real numbers.‚Ä∫

text‚ÄπWe define model of real numbers as any quadruple of sets $(K,A,M,r)$ 
  such that $(K,A,M,r)$ is an ordered field and the order relation $r$
  is complete, that is every set that is nonempty and bounded above in this 
  relation has a supremum.‚Ä∫

definition
  "IsAmodelOfReals(K,A,M,r) ‚â° IsAnOrdField(K,A,M,r) ‚àß (r {is complete})"

end
